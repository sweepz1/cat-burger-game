# 🐱 Purrsuit of the Burger 🍔

A retro pixel-art side-scrolling platformer where a very determined cat must navigate treacherous platforms, stomp enemies, and collect coins — all in pursuit of the ultimate prize: **a hamburger**.

## 🎮 How to Play

| Key | Action |
|-----|--------|
| `←` `→` | Move left / right |
| `Space` or `↑` | Jump (press again for double jump!) |
| Stomp enemies | Jump on top of them |

## 🏆 Scoring

| Action | Points |
|--------|--------|
| Collect a coin | +10 |
| Stomp an enemy | +100 |
| Grab the burger | +500 🍔 |

## 🕹️ Features

- Procedurally generated levels that get harder each round
- Double jump mechanic
- Enemy stomping
- Parallax scrolling pixel-art background with moon & stars
- Particle effects & floating score text
- Progress bar showing distance to the burger
- Lives system with invincibility frames
